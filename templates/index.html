IPL Match Winner Predictor

About the IPL Match Winner Predictor

This project uses historical IPL match data to predict the winner of an upcoming IPL match. It uses a machine learning model trained on data such as team performance, toss winner, and head-to-head matchups to make predictions.This is correct upto 2024

Key Features:

  • Predicts the winner of a match between two teams
  • Displays the last 5 head-to-head matches between teams
  • Shows the top-performing batsman and bowler in head-to-head matches
